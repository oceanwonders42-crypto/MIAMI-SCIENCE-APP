# Optional WebP (or other) exercise photos

Drop files here when using **`EXERCISE_PHOTO_PATH_OVERRIDES`** in `src/lib/exerciseMedia.ts`.

Suggested names (must match override paths):

- Men’s / default: `{slug}.webp` e.g. `bench-press.webp`
- Women’s: `{slug}-women.webp` e.g. `bench-press-women.webp`

If you do not use overrides, the app uses PNGs in the parent folder per `public/exercises/README.md`.
